## VGGFace-nnabla

### About
This repository is implementation of VGG-Face CNN with Sony's Neural Network Libraries, [NNabla](https://github.com/sony/nnabla).

### Requirement
- Python2
- NNabla 0.9.1rc3
- OpenCV 2.4.11
